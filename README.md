# Time Limits Config — Static Prototype

Static HTML/CSS/JS prototype for the **Time limits** configuration screen.

## Preview

Open `index.html` in a browser or run a local server:

```bash
cd /Users/alejandrosilva/Documents/time-limits-config
python3 -m http.server 8082
```

Then visit [http://localhost:8082](http://localhost:8082).

## Design

- PrimeNG lara-light-blue theme (CDN)
- PrimeIcons 7.0.0 (CDN)
- Vanilla JS + localStorage
- All UI text in English
